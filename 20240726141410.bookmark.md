Therefore, extracting data from Web sources has become a key technology. In this paper, we introduce a semi-automatic system <i>Ducky</i>: including a Web Wrapper which extracts data from Web sources and translates them into structured data. In <i>Ducky</i>, by defining a configuration file consisting in several parameters (URL of the Web page, CSS selectors which locates the data to retrieve and so on.), users do not need to write Web scraping programs at all. The definition is simple, yet can extract data flexibly from various structured Web pages. Additionally, <i>Ducky</i> provides a Web API and various output data formats: XML, JSON, CSV.
